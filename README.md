## Install Dependencies

**For Backend** - `cd server` `npm i --legacy-peer-deps`

**For Frontend** - `cd client` `npm i --legacy-peer-deps`

## Env Variables

**For Backend** - `cd server` `nodemon server.js`

**For Frontend** - `cd client` `npm run start`
